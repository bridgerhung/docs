四、蒐集目的

您的個人資料可能基於下列目的進行蒐集、處理及利用：

  3. 提供英文面試練習、錄音、語音播放及作答功能。
  4. 提供 AI 分析、學習提示、模擬問答及學習回饋。
  5. 保存學習歷程、作答內容、練習次數及完成進度。
  6. 供授課教師或經授權之助教進行課程評量、回饋及成績管理。
  7. 寄送帳號通知、密碼重設、作業提醒及其他必要系統通知。
  8. 維護系統正常運作、資訊安全、異常偵測、問題排查、備份及復原。
  9. 處理使用者詢問、系統錯誤回報、申訴、權益請求及相關行政作業。
  10. 配合校務、課程、計畫成果查核、稽核及依法應辦理的事項。
  11. 經另行依研究倫理審查核准方式辦理之研究參與者知情同意程序授權後，作為研究分析及成果發表之用。

  1. 一般帳號及課程資料：包含姓名、學號、班級、電子郵件、帳號狀態、作答文字、學習歷程、AI 回饋、教師評分、學習心得及其他課程相關資料，原則上保存至民國 116 年 9 月 1 日止。
  2. 錄音及相關音訊檔案：使用者於英文面試練習、作業或其他課程活動中產生或上傳的錄音及相關音訊檔案，原則上保存至民國 116 年 9 月 1 日止。
  3. 問題回報照片及附件：使用者因回報系統錯誤、功能異常、操作問題或提出技術支援需求所上傳的照片、螢幕截圖及其他附件，原則上保存至民國 116 年 9 月 1 日止。
  4. 系統操作及資訊安全紀錄：包含登入、登出、IP 位址、裝置資訊、Session、錯誤、人機驗證、安全稽核，以及練習期間進入或離開全螢幕、頁面隱藏、視窗失焦、貼上遭阻擋與異常大量輸入等操作紀錄，原則上保存至民國 116 年 9 月 1 日止。
  5. 研究資料：僅限經研究參與者知情同意程序授權納入研究使用的資料；其保存起訖、利用、退出及銷毀方式，依研究計畫、研究倫理審查核准內容及研究參與者知情同意內容辦理。
  6. 依法或處理爭議所必要的延長保存：如因法令、教育部、學校、研究倫理規範、課程申訴、成績爭議、資訊安全事件、司法調查或其他依法應辦事項而有繼續保存的必要，得於該特定目的所必要的期間內延長保存。

保存期限屆滿或蒐集目的消失後，相關資料將依資料性質及實際情況刪除、銷毀、匿名化或停止利用。但依法或依相關規定仍有保存義務者，不在此限。

（一）中華民國境內

使用者於中華民國境內使用本系統，以及經授權人員於執行課程教學、評量、研究計畫、行政管理、系統維護及個人資料權利處理等必要範圍內利用資料。

本系統主要使用 Oracle Cloud Infrastructure 美國西部聖荷西區域（us-sanjose-1），提供應用程式主機、資料庫、系統操作紀錄及其他系統運作所必要的雲端基礎設施服務。

本系統使用 Oracle Cloud Infrastructure Email Delivery 電子郵件傳送服務，處理寄件人及收件人的電子郵件地址、郵件主旨、郵件內容、寄送狀態、退信紀錄及郵件抑制紀錄等必要資料。

（三）日本東京都

本系統使用 Oracle Cloud Infrastructure 日本東京區域（ap-tokyo-1）的物件儲存服務，保存下列檔案：

  1. 使用者於英文面試練習、作業或其他課程活動中產生及上傳的錄音檔案。
  2. 使用者因回報系統錯誤、功能異常、操作問題或提出技術支援需求，而上傳至本服務的照片、螢幕截圖及其他相關附件。

上述檔案可能包含使用者的聲音、畫面內容、裝置畫面、帳號資訊或其他可識別個人身分的資料。

使用者於回報問題時，應避免上傳與問題排查無關的個人資料、帳號密碼、驗證碼、金融資料、醫療資料或其他敏感及機密資訊。

（四）Cloudflare Turnstile 全球服務節點所在地

本系統使用 Cloudflare Turnstile 提供人機驗證服務，以辨識自動化程式，並降低帳號遭濫用、惡意登入或大量自動化請求的風險。

使用者進行登入、密碼重設申請或其他需要人機驗證的操作時，其 IP 位址、瀏覽器及裝置資訊、連線時間、人機驗證結果及必要技術資訊，可能經由 Cloudflare 的全球服務節點進行即時處理。

（五）OpenAI API 及其受託處理者所在地

為提供錄音辨識、英文作答分析、草稿修正、學習提示、面試模擬、AI 回饋及文字轉語音等功能，本系統可能將執行功能所必要的錄音檔案、語音內容、作答文字、對話內容、系統提示及相關資訊傳送至 OpenAI API。

OpenAI Audio Transcriptions API以 whisper-1 模型進行錄音檔案的英文語音辨識、文字轉錄及必要的字級時間資訊分析。
OpenAI Chat Completions API處理英文作答分析、文法修正、草稿回饋、學習建議、面試模擬及其他文字內容生成；目前預設文字模型為 gpt-5.4-mini。
OpenAI Audio Speech API將系統提供的英文文字轉換為語音，供跟讀、示範或相關學習功能使用；目前使用 gpt-4o-mini-tts 模型，角色扮演語音使用 cedar，跟讀示範語音使用 marin。

相關資料可能於中華民國境外，包括美國、日本、歐洲及 OpenAI 現行次級處理者提供服務的其他地區進行處理。OpenAI 的受託服務提供者及處理地區可能隨服務調整，最新資訊請參閱其次級處理者清單。

除非本系統另行啟用特定的資料駐留或區域處理功能，相關資料處理不保證限定於中華民國境內或單一國家或地區。

本系統使用的生成式 AI、語音辨識及文字轉語音服務均由 OpenAI 提供，未使用其他非 OpenAI 的第三方 AI 或語音服務。

使用者於作答文字、錄音、照片、螢幕截圖或其他附件中自行提供姓名、學號、聯絡資訊或其他個人資料時，該等資訊可能隨檔案或內容傳送至上述服務所在地。因此，使用者不應提供與課程活動或問題排查無關的敏感或機密資料。

十、個人資料利用方式與瀏覽器儲存

  1. 建立帳號、身分驗證及權限管理。

必要的 Cookie 與瀏覽器儲存

本系統使用必要的登入及跨站請求防護 Cookie 維持身分驗證與系統安全；其中登入憑證採 HttpOnly Cookie，前端程式無法直接讀取。系統不使用廣告或行銷追蹤 Cookie。

為維持操作流程，瀏覽器的 sessionStorage 可能暫存使用者名稱、角色、登入狀態、導覽狀態及角色扮演關鍵字草稿；localStorage 可能暫存尚未送出的學習心得草稿及 PWA 安裝狀態。這些資料保存在使用者裝置，可透過登出、完成送出、關閉工作階段或清除瀏覽器網站資料移除；使用共用裝置時應於完成後登出並關閉瀏覽器。

十一、AI 及第三方服務處理

為提供錄音辨識、英文作答分析、草稿修正、面試模擬、學習回饋及文字轉語音功能，本系統會將執行功能所必要的錄音檔案、語音內容、作答文字、草稿、對話內容、系統提示及相關資訊傳送至 OpenAI API。

（一）錄音辨識

使用者的錄音檔案會傳送至 OpenAI Audio Transcriptions API，並使用 whisper-1 模型進行英文語音辨識及文字轉錄。

（二）文字分析及內容生成

語音辨識所得文字、使用者輸入的作答、草稿、對話內容及執行功能所必要的系統提示，會傳送至 OpenAI Chat Completions API，以進行：

  1. 英文作答分析。
  2. 文法及用字修正。
  3. 草稿回饋及範例生成。
  4. 面試問題及追問生成。
  5. 面試角色模擬。
  6. 學習建議及整體回饋。
  7. 將分析結果整理為系統所需的結構化資料。

系統使用的文字模型由系統設定決定，目前預設為 gpt-5.4-mini。模型可能因系統功能、效能、安全性或服務調整而變更。

（三）文字轉語音

為提供英文跟讀、發音示範或相關語音播放功能，系統提供的英文文字可能傳送至 OpenAI Audio Speech API，並使用 gpt-4o-mini-tts 模型產生 MP3 語音檔案。角色扮演語音目前使用 cedar，跟讀示範語音目前使用 marin；兩者依學習用途套用不同的語音指示。

文字轉語音所傳送的內容原則上為系統產生或課程功能所需的英文文字，不需要傳送使用者的原始錄音檔案。

（四）OpenAI 保存與模型訓練

OpenAI 對 API 輸入及輸出的保存依本系統 API 帳戶、專案及端點所啟用的資料控制設定辦理。未啟用零資料保留等核准控制時，OpenAI 依其現行企業隱私說明，API 輸入及輸出通常會於三十日內自其系統移除，但依法必須保存者不在此限。

OpenAI 表示 API 平台資料預設不會用於訓練其模型，除非帳戶持有人明確選擇加入資料分享。本計畫不會為模型訓練目的主動選擇分享學生資料。最新政策以 OpenAI 的企業隱私說明為準。

（五）資料最小化及使用者注意事項

本系統使用的生成式 AI、語音辨識及文字轉語音服務均由 OpenAI 提供，未使用其他非 OpenAI 的第三方 AI 或語音服務。

系統將依資料最小化原則，避免向 OpenAI 傳送與功能無關的姓名、學號、電子郵件或其他直接識別資料；但使用者自行於作答文字、草稿、對話或錄音中提及的資訊，仍可能包含於傳送內容中。

使用者不應於作答、草稿、對話或錄音中提供與課程無關的身分證字號、金融資料、醫療資料、密碼、驗證碼、私人通訊內容或其他敏感及機密資訊。

十二、當事人得行使的權利

依個人資料保護法，您得就本人資料請求：

  1. 查詢或閱覽。
  2. 製給複製本。
  3. 補充或更正。
  4. 停止蒐集、處理或利用。
  5. 刪除。

您可以透過本頁第十五節所列的計畫主持人電子郵件、電話或其他經學校公告的管道提出申請。提出申請時，可能需要提供足以確認本人身分及請求範圍的資料；查詢、閱覽或製給複製本時，得依法酌收必要成本費用。

查詢、閱覽或製給複製本之請求，原則上於十五日內為准駁決定；補充、更正、停止蒐集／處理／利用或刪除之請求，原則上於三十日內為准駁決定。依法需要延長時，將於原期限內說明原因並通知請求人。

如依法、課程評量、申訴處理、資訊安全或法定保存義務仍有保存必要，部分請求可能無法立即辦理，屆時將說明理由。

上述權利不得預先拋棄或以特約限制。

十三、提供資料的自由及不提供的影響

您可以決定是否提供非課程或系統運作所必要的資料。

但如不提供帳號識別、班級、電子郵件、作答、錄音或教師評量所必要的資料，可能產生以下影響：

  1. 無法建立或驗證帳號。
  2. 無法使用相應的作答、錄音、AI 回饋或教師評量功能。
  3. 無法完成授課教師指定的課程活動。
  4. 無法接收帳號、密碼重設或作業相關通知。

是否參與研究，由您另行透過研究倫理審查核准方式辦理之研究參與者知情同意程序決定。

不參與研究本身不影響您的課程成績、受教權益或與授課教師之關係；但仍應依課程要求完成一般教學活動及指定作業。

十四、資料安全

系統將採取合理的技術及管理措施保護個人資料，包括：

  1. 帳號權限及身分驗證。
  2. 密碼雜湊儲存及具適當安全強度之傳輸加密。
  3. 存取控制及操作紀錄。
  4. 備份、監控及資訊安全防護。
  5. 限制經授權人員於必要範圍內存取資料。
  6. 發生個人資料事故時，依相關規定進行處理及通知。

惟網路、跨境傳輸及資訊系統無法保證絕對零風險，使用者亦應妥善保管帳號密碼，避免於共用或不可信任裝置登入。
