本系統由國立雲林科技大學通識教育中心之教育部教學實踐研究計畫團隊建置，提供英文面試練習、作答錄音、AI 輔助回饋、學習歷程紀錄及教師評量等功能。

使用本系統前，請詳閱本條款及個人資料蒐集、處理及利用告知事項。本條款的確認不代表同意參與研究。

一、適用範圍與使用目的

本系統僅供經授權之學生、教師及相關工作人員進行課程教學、英文面試練習、學習回饋、教師評量、系統維護及相關行政作業使用。

使用者不得將本系統用於與課程無關、違反法令、侵害他人權益或妨礙系統正常運作之用途。

本條款依中華民國法律、校規及課程規定適用與解釋，不影響使用者依法或依學校程序享有的申訴及其他權利。

二、帳號及密碼安全

  1. 使用者應使用本人帳號登入，不得冒用他人身分、借用或轉讓帳號，亦不得代替他人完成課程活動。
  2. 使用者應妥善保管帳號、密碼及登入連結，不得提供予他人使用。
  3. 使用者首次登入時，應立即更換系統提供的初始密碼。
  4. 如發現帳號遭冒用、密碼外洩或其他異常情形，應儘速通知授課教師或系統管理人員。
  5. 因使用者故意或重大過失，造成帳號遭不當使用或系統及他人權益受損者，使用者可能須承擔相應責任。

三、作答、錄音及上傳內容

  1. 使用者於本系統輸入、錄製或上傳的文字、聲音、照片、螢幕截圖及其他資料，應以課程學習、指定活動或系統問題回報為限。
  2. 使用者不得輸入或上傳違法、侮辱、歧視、騷擾、威脅、侵害隱私、侵害著作權或其他侵害第三人權益的內容。
  3. 未經他人同意，不得錄製、上傳或散布含有他人聲音、姓名、影像或其他可識別資訊的內容。
  4. 使用者應確認其提交內容未含與課程或問題排查無關的機密資料、帳號密碼、驗證碼、金融資訊、醫療資訊或其他不宜提供的敏感資料。
  5. 使用者仍保有其自行創作內容依法享有的權利；確認本條款不會將著作權移轉給系統團隊。
  6. 為提供課程教學、評量、備份、系統運作及技術支援，使用者同意系統於必要範圍及保存期間內儲存、重製、傳輸及顯示其提交內容。將內容納入研究分析，仍須另依研究參與者知情同意程序辦理。

四、AI 輔助功能

  1. 本系統部分功能使用人工智慧技術分析作答內容，並產生提示、建議、模擬問答或學習回饋。
  2. AI 產生的內容可能不完整、不精確、不適合個別情境，亦可能產生錯誤資訊。
  3. AI 回饋僅供學習參考，不構成正式專業意見、就業保證或教師最終評量。
  4. 正式課程要求、成績及評量結果，仍以授課教師或經授權評分人員的認定為準。
  5. 使用者不得利用提示注入、自動化工具或其他方式，試圖取得系統指令、密鑰、其他使用者資料或未經授權的資訊。

五、禁止行為

使用者不得從事下列行為：

  1. 冒用、竊取或未經授權使用他人帳號。
  2. 以自動化程式大量發送請求、惡意消耗系統資源或規避使用限制。
  3. 掃描、測試、攻擊、干擾或破壞系統、伺服器、網路及安全機制。
  4. 嘗試存取其他使用者的作答、錄音、成績、個人資料或管理功能。
  5. 未經授權對正式服務進行反向工程、破解、繞過權限，或取得未公開的程式、模型指令或系統資訊。但依法或依適用的開放原始碼授權條款所允許之使用，不在此限。
  6. 將系統產生的內容用於作弊、冒名作答或其他違反學術倫理的行為。
  7. 從事任何違反法令、校規、課程規定或侵害他人權益的行為。

六、系統可用性與資料保存

  1. 本系統可能因維護、更新、安全處理、第三方服務異常、網路中斷或不可抗力而暫停或限制部分功能。
  2. 系統團隊將合理維護系統安全及穩定性，但不保證服務永不中斷，亦不保證所有內容均可永久保存。
  3. 使用者應依課程要求，視需要自行保存重要作答或成果。
  4. 如發生資料毀損、服務中斷或其他異常，系統團隊將於合理可行範圍內進行處理。

七、違規處理

如使用者違反本規範、影響系統安全或侵害他人權益，授課教師或系統管理人員得視情況採取下列措施：

  1. 提醒或要求停止相關行為。
  2. 限制部分系統功能。
  3. 暫停或停用帳號。
  4. 保留必要的操作及安全紀錄。
  5. 通知授課教師、校方相關單位，或依法配合主管機關處理。

涉及緊急資訊安全、他人權益或證據保存時，系統團隊得先採取必要的暫時限制；其他情形原則上會以適當方式告知原因及後續處理方式。

使用者對帳號限制、違規認定、課程評量或其他處理有疑義時，可向授課教師提出說明或依學校申訴程序辦理。系統條款不取代校方既有的成績申訴、學生權益或學術倫理程序。

八、研究參與說明

本系統可能用於教育部教學實踐研究計畫相關課程活動。

是否參與研究，以及哪些資料得作為研究資料使用，均以另行依研究倫理審查核准方式辦理之「研究參與者知情同意」內容為準。

本條款的確認不等同於同意參與研究，亦不取代研究參與者知情同意程序。

九、條款修訂

因應法令、課程內容、系統功能、服務供應商或資訊安全需求，本條款可能進行修訂。頁面會標示版本、發布日期及生效日期。

如修訂內容對使用者權益有重大影響，系統將以適當方式重新通知，並要求使用者分別確認新版本。系統會保留各次確認的條款版本及時間。
